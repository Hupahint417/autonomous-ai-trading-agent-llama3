# Contributing to Autonomous AI Trading Agent

We welcome contributions to the public beta. Here are the guidelines.

## How to Contribute

1. **Fork** the repository
2. **Create a branch**: `git checkout -b feature/your-feature`
3. **Make your changes** — ensure code quality and tests pass
4. **Commit** with clear messages
5. **Push** and open a Pull Request

## What We Welcome

- New exchange connectors (via CCXT)
- Indicator modules
- Prompt improvements for the reasoning engine
- Bug fixes and documentation

## Code Style

- Python 3.10+
- Follow PEP 8
- Add type hints where appropriate

## Before Submitting

- Run the agent in paper mode to verify your changes
- Do not include API keys or secrets
