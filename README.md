## ⚙️ How the Bot Works

The bot uses AI to analyze multiple data points:

- **RSI (Relative Strength Index):** Measures market momentum to detect overbought or oversold conditions.
- **MACD (Moving Average Convergence Divergence):** Tracks trend changes and momentum.
- **Live News:** Analyzes recent news events that might affect crypto prices.
- **Chain-of-thought reasoning:** The AI processes these signals in steps to come to a conclusion before trading.

Because the AI runs locally on your machine, your data never leaves your computer. This improves security and privacy.

The bot connects to over 100 crypto exchanges through CCXT, a widely used trading API library. This means you can trade on Binance, Bybit, and many other platforms.

## 🛠️ Setting Up API Keys

To let the bot trade automatically, you need to create API keys from your exchange account.

Here’s how to do that:

1. Log into your crypto exchange account.
2. Navigate to the API management section.
3. Create a new API key with permissions to trade (do not enable withdrawal permissions).
4. Copy the API key and secret.
5. Paste them into the bot’s setup screen.

Keep your API keys private and never share them with others.
